True Cross-Platform

Powered by Haxe, Kha can cross-compile your code and optimize your assets for even the most obscure systems. Build your applications natively to desktops, tablets, phones and consoles. Implement once, run everywhere: Windows, macOS, Linux, HTML5, iOS, Android, Windows Phone, tvOS, Tizen, Flash, Raspberry Pi, Xbox One and even Unity 3D.

Kode Studio

Kha has its own IDE. Available for Windows, macOS and Linux, it has powerful features as IntelliSense, built-in Git and debugging right from the editor.
